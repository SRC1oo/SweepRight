# SweepRight Cleaning

Production-ready booking website for SweepRight Cleaning.

## Overview
- Instant quote builder for home and commercial cleaning
- Time-based pricing and add-ons
- Full payment via Stripe Checkout
- Booking confirmation emails for customer and internal team
- Hosted on Cloudflare Workers with static assets + backend

## Tech
- Cloudflare Workers
- Wrangler (`wrangler.jsonc`)
- Stripe Checkout
- Basic email backend
